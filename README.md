# Excel Report Bot

Telegram-бот для автоматической бизнес-аналитики из Excel-файлов.

**Стек:** Python 3.11+ · aiogram 3.x · pandas · APScheduler · aiosqlite · FastAPI · loguru · matplotlib

---

## Что умеет

- Парсит `.xlsx` файлы с продажами (автодетект колонок, совместимость с 1С-выгрузками)
- Считает выручку, средний чек, ТОП-5 товаров, критические остатки
- **ABC-анализ** — автоматически делит товары на классы A/B/C по доле в выручке
- Три режима отчёта: **Кратко** / **Полный** / **С графиком**
- **Два графика** в режиме "С графиком": топ товаров + динамика выручки по дням
- `/compare` — сравнение двух последних периодов с дельтой ▲/▼, выручкой в день и категориями
- `/export` — скачать полный отчёт как `.txt` файл
- Авторассылка по расписанию (каждый пользователь выбирает своё время и часовой пояс)
- **Алерт о нулевых остатках** при авторассылке — отдельное срочное сообщение
- История последних 10 отчётов с возможностью повтора
- Роли: `admin` (полный доступ) / `viewer` (только просмотр)
- FastAPI `/health` эндпоинт для мониторинга

## Формат Excel-файла

Минимально необходимые колонки (регистр не важен):

| Товар | Количество | Сумма | Остаток | Категория | Дата |
|-------|-----------|-------|---------|-----------|------|
| Хлеб  | 10        | 500   | 15      | Еда       | 2026-04-01 |

**Альтернативные названия колонок:**
- Товар: `name`, `item`, `наименование`, `продукт`
- Количество: `qty`, `count`, `кол-во`
- Сумма: `amount`, `revenue`, `выручка`, `продажи`
- Остаток: `stock`, `balance`, `остатки`
- Категория: `category`, `group`
- Дата: `date`, `день`

## Режимы отчёта

| Режим | Содержание |
|-------|-----------|
| **Кратко** | Выручка, позиции, средний чек, топ-3 товара, остатки |
| **Полный** | Топ-5 товаров, категории, остатки, **ABC-анализ** |
| **С графиком** | Полный отчёт + график топ товаров + **график выручки по дням** |

## Пример отчёта (режим "Кратко")

```
📊 Отчёт за 01.04–05.04.2026

💰 Выручка: 1 234 567 ₽
📦 Позиций: 342
🧾 Средний чек: 3 610 ₽

🏆 ТОП товары:
  1. Товар А — 234 500 ₽ (19%)
  2. Товар Б — 187 200 ₽ (15%)
  3. Товар В — 143 800 ₽ (12%)

⚠️ Заканчивается (3 поз.): Товар Х, Товар У, Товар З
🚫 Нет в наличии (2 поз.): Товар М, Товар Н
```

## Пример ABC-анализа (в режиме "Полный")

```
🔬 ABC-анализ

🅐 Класс A (3 поз.) — 80% выручки:
  Товар А, Товар Б, Товар В

🅑 Класс B (5 поз.) — след. 15%:
  Товар Г, Товар Д, Товар Е...

🅒 Класс C — 14 позиций (оставшиеся 5%)
```

## Пример сравнения периодов

```
📈 Сравнение периодов

Период 1: 01.03.2026–31.03.2026
Период 2: 01.04.2026–30.04.2026

💰 Выручка:
  937 420 ₽ → 1 112 153 ₽  ▲ 18.6%

📅 Выручка в день:
  30 239 ₽ → 37 072 ₽  ▲ 22.6%

📂 Категории:
  Напитки: 168 300 ₽ → 198 500 ₽  ▲ 17.9%
  Мясо:    157 000 ₽ → 188 000 ₽  ▲ 19.7%
```

## Деплой на VPS

